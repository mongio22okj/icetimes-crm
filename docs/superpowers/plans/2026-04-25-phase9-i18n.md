## Task 1 — Config + tagging

- Add `LANGUAGES`, `LOCALE_PATHS` to settings
- Add `LocaleMiddleware` to MIDDLEWARE (after SessionMiddleware)
- Add `path("i18n/", include("django.conf.urls.i18n"))` to apex/urls
- Wrap `NAV_ITEMS` labels in `gettext_lazy`
- Add `{% load i18n %}` and `{% trans %}` tags to: header.html, sidebar.html, dashboard/index.html, login.html, marketing/base.html
- Run `makemessages -l es` to generate `locale/es/LC_MESSAGES/django.po`

**Commit:** `feat(i18n): config + tag sidebar nav and chrome templates`
