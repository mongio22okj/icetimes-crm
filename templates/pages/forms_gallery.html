Combobox

Single-select typeahead. Static option list shown here; pass async_url to load options from an HTMX/JSON endpoint backed by TypeaheadMixin.

{% apex_field demo_choice.owner %}
owner = forms.ModelChoiceField(
    queryset=User.objects.all(),
    widget=Combobox(async_url=reverse_lazy("users:typeahead")),
)

Date range picker

Trigger button + popover with two native date inputs and preset shortcuts. Stores the range as a comma-joined ISO string.

{% apex_field demo_date_upload.period %}
period = forms.CharField(
    required=False,
    widget=DateRangePicker(),
)

def clean_period(self):
    raw = self.cleaned_data["period"] or ""
    parts = [p.strip() for p in raw.split(",")]
    if len(parts) == 2 and parts[0] and parts[1]:
        from datetime import date
        return (date.fromisoformat(parts[0]), date.fromisoformat(parts[1]))
    return None

File dropzone

Drag-drop multi-file uploader with previews and per-file XHR progress. The widget POSTs each file to upload_url and tracks the returned IDs in the hidden field value.

{% apex_field demo_date_upload.attachments %}
attachments = forms.CharField(
    required=False,
    widget=FileDropzone(
        upload_url="files:upload_xhr",   # URL name or path
        accept="image/*,.pdf",
        max_files=5,
        max_size_mb=10,
    ),
)

Character counter

Standalone Alpine helper. Wrap any input + counter element with x-data="apexCharCounter(maxLen)"; the counter classes auto-color amber → red as you approach the cap.

/ 

<div x-data="apexCharCounter(140)">
  <textarea x-ref="input" @input="update($event.target.value)"
            maxlength="140"></textarea>
  <p :class="classes">
    <span x-text="n"></span> / <span x-text="max"></span>
  </p>
</div>
