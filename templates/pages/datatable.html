What's included

  • • Server-side everything. Sort, filter, search, paginate, bulk action, export — all via the URL.
  • • HTMX swaps. Every interaction updates just the table region; no full page reload.
  • • Bulk actions. Row checkboxes + select-all-on-page + Alpine-powered confirm modal.
  • • Saved views. Per-user named filter+sort combos with default-view auto-apply.
  • • Column visibility. Per-user persisted; pinned columns stay locked.
  • • 3 export formats. CSV (stdlib), XLSX (openpyxl), PDF (WeasyPrint, capped 500 rows).
  • • 6 filter widgets. Text, select, multi-select, daterange, numeric range, boolean.
  • • Multi-sort. Shift-click to add secondary sort columns.
  • • Mobile-friendly. Table collapses to stacked cards under md.
  • • Empty states. Distinct copy for "no data yet" vs "no matches".

Adding a table to your view

# apps/widgets/views.py
from apps.core.tables import BulkAction, Column, Filter, TableConfig, TableView

WIDGETS_TABLE = TableConfig(
    key="widgets",                         # unique; scopes prefs + saved views
    columns=(
        Column("name", "Name", searchable=True, pinned=True),
        Column("status", "Status",
               filter=Filter("select", choices=Widget.STATUS)),
        Column("created_at", "Created",
               filter=Filter("daterange"),
               formatter=lambda v: v.strftime("%b %d, %Y") if v else ""),
    ),
    bulk_actions=(
        BulkAction("archive", "Archive", icon="trash", destructive=True,
                   confirm_text="Archive {n} widgets?"),
    ),
    default_sort="-created_at",
    page_size=25,
    empty_headline="No widgets yet",
    empty_body="Create your first widget to get started.",
)


class WidgetListView(LoginRequiredMixin, TableView):
    model = Widget
    template_name = "widgets/list.html"   # extends layouts/dashboard.html
    table_config = WIDGETS_TABLE

    def handle_bulk_action(self, action, ids, request):
        if action.slug == "archive":
            Widget.objects.filter(pk__in=ids).update(archived=True)
        return redirect("widgets:list")
