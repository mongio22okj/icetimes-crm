{# ── Webhooks ──────────────────────────────────────────── #}

Webhooks

Subscribe to events to receive real-time notifications. Apex signs every payload with HMAC-SHA256 — verify the X-Apex-Signature header before processing.

Sample event types

  • customer.created
  • customer.archived
  • order.placed
  • invoice.sent
  • invoice.paid
  • subscription.canceled

{# ── Errors ────────────────────────────────────────────── #}

Errors

Apex uses standard HTTP status codes. Error bodies always include code and message.

Status Code Meaning
200okSuccess
400invalid_requestMalformed body or missing required field
401unauthenticatedMissing or invalid API token
403forbiddenToken lacks the required scope
404not_foundResource doesn't exist
409conflictStale write — refetch and retry
429rate_limitedSlow down; see Retry-After header
5xxinternal_errorApex bug or downstream outage
{# ── Rate limits ───────────────────────────────────────── #}

Rate limits

Each token is rate-limited per minute. The current window is exposed in response headers.

Free

60 /min

Pro

600 /min

Enterprise

3,000 /min

X-RateLimit-Limit · X-RateLimit-Remaining · X-RateLimit-Reset
