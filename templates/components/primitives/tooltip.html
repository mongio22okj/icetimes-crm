{% comment %} Reusable Tooltip primitive — wraps any trigger element. Use the include only for the simplest case (text-only content). For richer triggers, copy the markup inline since {% include %} can't carry block content. Args: text required — tooltip body side top | right | bottom | left (default top) trigger_text text shown for the trigger (when using the include) trigger_class CSS classes for the trigger button (default: pill style) {% endcomment %} {% with side=side|default:"top" %} {{ text }} {% endwith %}
