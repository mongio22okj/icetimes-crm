Open variants

Each trigger opens a drawer from a different edge. Esc closes; backdrop click closes.

Usage

{% templatetag openblock %} include "components/primitives/drawer.html" with id="my-drawer" title="Filters" side="right" body="…content…" {% templatetag closeblock %}

<button @click="$dispatch('apex:open', 'my-drawer')">Open</button>

Accessibility

  • • role="dialog" + aria-modal="true" + labelled-by the title.
  • • Esc and backdrop-click close the panel.
  • • Focus restores to the triggering element on close (apexDrawer() factory).
  • • Slide animations gated on prefers-reduced-motion.
