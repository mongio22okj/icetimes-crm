Usage

  <button @click="toggle('a')" :aria-expanded="isOpen('a')">…</button>
  <div x-show="isOpen('a')" x-cloak role="region">…</div>

Accessibility

  • • Trigger uses aria-expanded + aria-controls.
  • • Region uses role="region".
  • • Open/close animates via Alpine's x-collapse when the plugin is loaded; falls back to x-show otherwise.
