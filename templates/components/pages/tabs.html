Usage

<div x-data="apexTabs('overview')">
  <div role="tablist" aria-label="Sections" class="flex border-b border-border">
    <button role="tab" data-tab="overview"
            :aria-selected="active==='overview'"
            :tabindex="active==='overview' ? 0 : -1"
            @click="set('overview')"
            @keydown.right.prevent="next()"
            @keydown.left.prevent="prev()"
            …>Overview</button>
    {# repeat for each tab; data-tab must match what set() expects #}
  </div>
  <div role="tabpanel" x-show="active==='overview'">…</div>
</div>

Accessibility

  • • Tablist uses role="tablist" + aria-label (or aria-orientation="vertical").
  • • Each tab uses role="tab" + aria-selected + roving tabindex (only the active tab is in the tab order).
  • • ←/→ cycle horizontal tabs; ↑/↓ cycle vertical; Home/End jump to first/last.
  • • Each panel uses role="tabpanel".
