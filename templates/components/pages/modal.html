Open variants

Click any trigger to open the matching modal. Esc closes; backdrop click closes; focus returns to the trigger.

{# ── Markup snippet ───────────────────────────────────────────── #}

Usage

Drop the include into any template; trigger from anywhere via Alpine's $dispatch.

{% templatetag openblock %} include "components/primitives/modal.html" with id="my-modal" title="Confirm" body="Sure?" primary_label="Yes" secondary_label="Cancel" {% templatetag closeblock %}

{# Trigger from anywhere: #}
<button @click="$dispatch('apex:open', 'my-modal')">Open</button>
{# ── Accessibility notes ──────────────────────────────────────── #}

Accessibility

  • • role="dialog" + aria-modal="true" + labelled-by the title.
  • • Esc closes the modal.
  • • Backdrop click closes; clicks inside the panel do not.
  • • Focus returns to the triggering element on close (handled by apexModal() factory).
  • • Animations are gated on prefers-reduced-motion via Alpine transitions.
