Guida

Come funziona il CRM, passo per passo.

1. Accesso

Login su /accounts/login/ con utente admin. Dopo il login entri nel CRM (sidebar a sinistra).

2. Com'è organizzato

  • CRM → dashboard economica (guadagni, spese, profitto, FTD)
  • Dashboard → riepilogo lead (KPI + lead recenti)
  • Lead → tabella completa dei lead con filtri
  • Broker API → configurazione dei broker
  • Account → Administration, Users, Settings, Guida

3. Configurare un broker

Sidebar → Broker API. Aggiungi con + TrackBox, + IREV o + SPM Monster (in base al tipo di API del broker).

Compili: nome, base_url, credenziali (chiavi/token), parametri (ai/ci/gi · affiliate/offer · affc/bxc/vtc), funnel, slug landing, brand landing, eventuale HTML landing, nota.

Le credenziali restano interne. Il brand landing è ciò che vede il visitatore; il nome interno (es. "Link Special 9") lo vedi solo tu nel CRM.

4. Le landing

Ogni broker ha la sua landing: /lp/<slug>/.

  • Landing nostra: mandi traffico al link /lp/<slug>/ (mostra brand + form).
  • Landing esterna (tua): in Broker API → Codice copi lo snippet <form> e lo incolli nel bottone "Inizia ora" della tua landing.

5. Il flusso di un lead

  1. Il visitatore compila la landing e invia.
  2. Il CRM crea il Lead con un click_id univoco.
  3. Il lead viene inviato (push) al broker via API.
  4. Il broker accetta → restituisce id + URL auto-login → il visitatore viene reindirizzato sul broker.
  5. Il lead compare nella pagina Lead.

I test reali li fai tu compilando la landing (i broker bloccano i bot).

6. Aggiornamento stati / FTD

  • TrackBox e IREV → via postback (il broker richiama il CRM in automatico).
  • SPM Monster → via pull/sync.
  • Sincronizzazione automatica ogni 10 minuti; oppure bottone "Aggiorna lead" sulla Dashboard per farlo subito.

✅ Quando un lead diventa FTD, la sua riga diventa verde in tutte le tabelle.

7. Pagina Lead

Tabella con tutte le colonne e una riga di filtri sotto le intestazioni: nome, email, telefono, IP, Paese (IT/ES/DE/SE), stato, broker, Depositare (Sì/No), ID cliccato. Premi Filtra / Reset.

9. Ciclo completo di una campagna

  1. Broker API → crea/scegli il broker, imposta slug + brand.
  2. Prendi il link landing /lp/<slug>/ oppure il Codice per la tua landing esterna.
  3. Mandi traffico.
  4. I lead arrivano in Lead, vengono inviati al broker, i visitatori vanno sul broker.
  5. Gli FTD si aggiornano da soli (riga verde).
  6. Controlli i guadagni in CRM (per broker, per settimana).
