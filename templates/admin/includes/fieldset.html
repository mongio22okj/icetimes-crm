{% comment %} Apex override of Django's admin fieldset.html. Identical to the stock template (Django 6) except every {{ field.label_tag }} is filtered through |cut:":" to strip the trailing-colon convention. Modern admin themes don't use the "Name:" suffix; we get cleaner labels everywhere (change form, popup, inlines) without touching individual ModelAdmins. Risk: |cut:":" removes ALL colons in the label HTML — affecting label text that legitimately contains ":" (e.g. verbose_name="Time: HH:MM"). Practically rare; acceptable trade-off for not maintaining a Python hook on every form. {% endcomment %}
