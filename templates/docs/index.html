What is Apex Django?

Apex Django is a feature-complete admin dashboard template designed for teams that ship product instead of rebuilding boilerplate. Five dashboard variants, multi-tenant Organizations with RBAC, realtime notifications via Django Channels, a full Django Ninja API, PDF invoicing, settings depth that matches Stripe's dashboard, and a public marketing site — all server-rendered with Tailwind v4, Alpine.js, and HTMX. No React, no bundle pipeline beyond Tailwind, no SPA.

Quick start

Five commands, ~5 minutes. SQLite + the in-memory channel layer cover everything in dev — no Postgres, no Redis required.

{% code lang="bash" %}git clone https://github.com/puikinsh/dashboardpack-apex-django.git cd dashboardpack-apex-django uv sync --all-groups npm install && npm run build uv run python manage.py migrate uv run python manage.py seed_demo uv run python manage.py runserver{% endcode %}

Open http://localhost:8000/ and sign in with the auto-filled demo / ApexShowcase!2026.
