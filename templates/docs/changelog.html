0.18.0 — Apex-themed Django Admin

Django's /admin/ is now fully re-skinned to match the rest of Apex — same sidebar, header, dark mode, command palette, toast notifications, OKLCh colour tokens. New apex.admin.ModelAdmin base class with show_in_dashboard + apex_icon hooks for opt-in stat cards and Lucide icons. Themed login, password change, change list with empty states, change form with proper label/input grid, delete confirmation, history, popup windows for related-object add/edit, file inputs with thumbnail preview for images, Select2 autocomplete, M2M filter_horizontal selector, calendar popup with Lucide chevrons, date/time shortcut chips, 404 / 500 error pages, and parity coverage of every other stock Django admin feature. Existing admin.ModelAdmin registrations pick up the theme automatically via template overrides.

0.17.2 — Higher-contrast dark mode

Reworked the .dark palette in static_src/css/input.css to mirror the Apex Next.js reference: pure neutral grays (chroma 0) instead of the subtly teal-tinted (hue 170) grays, with bigger lightness steps between background → card → muted → border. Cards now visibly lift off the background and input borders read clearly. Background L 0.09 → 0.075, card 0.125 → 0.19, popover → 0.21, secondary / muted / accent 0.19 → 0.235, border / input 0.20 → 0.28. Sidebar 0.075 → 0.11, sidebar-accent 0.14 → 0.20, sidebar-border 0.16 → 0.26. Light mode untouched.

0.17.1 — Microsoft SQL Server support

Apex now officially supports SQL Server 2017+ as a production database alongside Postgres. Postgres remains the default. Install with uv sync --extra mssql and set DATABASE_URL to an mssql:// or sqlserver:// scheme.
