Run the suite

{% code lang="bash" %}uv run pytest # unit tests, ~90s uv run pytest -m e2e # Playwright E2E (spawns Chromium) ./run-e2e.sh # both, sequentially{% endcode %}

The unit suite is the default; E2E is excluded by default (addopts = "-m 'not e2e'" in pyproject.toml).

Test layout

Tests live under each app:

{% code lang="text" %}apps//tests/ ├── factories.py # factory-boy factories ├── test_models.py ├── test_views.py └── test_forms.py{% endcode %}

Cross-cutting tests (errors, navigation, smoke, seed_demo, health, throttle) live in apps/core/tests/. E2E tests live in tests/e2e/.

Factories

factory-boy factories in each app's tests/factories.py are also imported by seed_demo to populate the demo DB — keep them realistic.

{% code lang="python" %}# Example: apps/customers/tests/factories.py import factory from apps.customers.models import Customer class CustomerFactory(factory.django.DjangoModelFactory): class Meta: model = Customer name = factory.Faker("company") email = factory.LazyAttribute(lambda o: f"{o.name.lower().replace(' ', '')}@example.com"){% endcode %}

Async tests (WebSocket consumers)

Channels' WebsocketCommunicator speaks ASGI directly — no HTTP server required. The in-memory channel layer (settings default in dev/test) handles fan-out so we don't need Redis.

{% code lang="python" %}import pytest from channels.testing import WebsocketCommunicator from apps.realtime.consumers import NotificationConsumer @pytest.mark.django_db(transaction=True) @pytest.mark.asyncio async def test_notification_consumer_accepts_authenticated_user(): user = await _make_user(...) comm = WebsocketCommunicator(NotificationConsumer.as_asgi(), "/ws/notifications/") comm.scope["user"] = user connected, _ = await comm.connect() assert connected is True await comm.disconnect(){% endcode %}

E2E tests (Playwright)

Playwright runs a real Chromium against live_server with the demo DB. Tests re-seed per-test via tests/e2e/conftest.py (transactional_db fixture).

{% code lang="bash" %}uv run pytest -m e2e tests/e2e/test_components.py{% endcode %} {% callout type="note" %} E2E tests set DJANGO_ALLOW_ASYNC_UNSAFE=true because Playwright runs an event loop, which trips Django's sync guard. {% endcallout %}

Single-test runs

{% code lang="bash" %}uv run pytest apps/orders/tests/test_views.py uv run pytest apps/orders/tests/test_views.py::TestName::test_method uv run pytest -k "not test_invoice" # skip a class of tests{% endcode %}
