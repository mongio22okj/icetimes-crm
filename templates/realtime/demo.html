How it works

  • {% icon "check" 14 %} Channels 4 + Daphne — ASGI replaces gunicorn for WebSocket support. HTTP requests still go through Django's normal stack.
  • {% icon "check" 14 %} In-memory channel layer in dev (zero infra). Set REDIS_URL + install the realtime extra in prod for cross-process fan-out.
  • {% icon "check" 14 %} Per-user fan-out via channel groups (notify.user.<id>). The notification dispatcher pushes to the layer after every Notification.objects.create().
  • {% icon "check" 14 %} Auto-reconnect with exponential backoff on the client. Anonymous connections close with code 4401 and don't retry.
